FAQ

전체 항공권 온라인체크인 수하물
공항에서 기내에서 홈페이지 기타
부가서비스 온라인면세점

Q [항공권] 혼합 결제 (무통장입금+카드)된 경우 홈페이지에서 취소가 가능한가요?

무통장입금은 혼합 결제 시 취소가 불가합니다. 단, 카드로 결제한 부가서비스는 따로 취소 가능합니다.

Q [항공권] 무통장입금 항공권 발권 완료 후 홈페이지에서 부가서비스 구매 카드 결제 시 부가서비스 쿠폰 사용은 가능한가요?

네, 부가서비스 구매 카드 결제 시에는 부가서비스 쿠폰 사용 가능합니다.

Q [항공권] 무통장입금 항공권 발권 완료 후 추가로 홈페이지를 통해 부가서비스구매 가능한가요?

네, 추가 구매는 가능하지만 결제 수단은 카드로만 가능하며 무통장입금은 불가합니다.

Q [항공권] 예약과 동시에 부가서비스를 추가하여 무통장입금 결제 완료 후 부가서비스만 취소 가능한가요?

네, 부가서비스만 취소 가능합니다.

Q [항공권] 예약과 동시에 부가서비스를 추가하여 무통장입금으로 결제가 가능한가요?

네, 항공권과 부가서비스 동시 구매 시 무통장입금으로 결제가 가능합니다.

Q [항공권] 무통장 입금 서비스 이용 시 취소수수료 적용 시점은 어떻게 되나요?

무통장 입금 서비스 이용 시 취소/변경 수수료의 적용은 예약 생성 시점을 기준으로 적용됩니다. 예약 결제 시점과 무관하게 최초 예약을 생성한 시점을 기준으로 적용되오니 이용에 참고하여 주시기 바랍니다.

Q [항공권] 무통장 입금 서비스 이용 시 쿠폰을 적용할 수 있나요?

무통장 입금의 경우 쿠폰 적용이 불가합니다.

Q [항공권] 무통장 입금 전 예약한 항공권을 취소할 수 있나요?

네, [나의 예약] ▷ [예약 상세 페이지]▷ [입금 전] 페이지에서 '결제 전 예약 취소' 버튼을 클릭하시면 무통장 입금 전 예약한 항공권 (결제 대기 중) 취소가 가능합니다.

Q [항공권] 무통장 입금 후 현금 영수증은 어떻게 발급 받을 수 있나요?

현금영수증 발급을 원하시는 경우 탑승 완료 후 당사 예약센터 1688-8686 (월~일, 07:00~19:00) 으로 문의하여 주시기 바랍니다.

Q [항공권] 무통장 입금 결제를 완료하면 항공권 발권이 바로 완료 되나요?

가상 계좌로 결제 금액을 입금하신 후 1시간 내로 예약한 항공권의 발권이 완료됩니다. 예약 발권 완료 시 남겨주신 핸드폰 번호로 알림톡이 발송되며 예약내역 [나의 예약] ▷[정상] 에서 확인 가능합니다.

아직 궁금증이 해결되지 않았나요?
