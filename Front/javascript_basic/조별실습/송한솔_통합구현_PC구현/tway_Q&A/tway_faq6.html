FAQ

전체 항공권 온라인체크인 수하물
공항에서 기내에서 홈페이지 기타
부가서비스 온라인면세점

Q [항공권] 편의점에서 결제 전 취소하고 싶습니다.

편의점 결제 전 취소는 [마이페이지>예약관리>항공권예약관리>결제전/확인필요 탭]에서 취소가 가능합니다.

Q [항공권] 편의점결제는 부가서비스 구매가 가능한가요?

항공권 예매 시 부가서비스를 선택하신 경우 부가서비스를 포함한 금액을 편의점에서 결제하실 수 있습니다. 하지만 편의점에서 대금 납부 후 [마이페이지>예약관리>항공권예약관리] 화면에서 부가서비스 추가 구매 시에는 편의점결제 이용이 불가합니다. 편의점에서 대금 납부 후 [마이페이지>예약관리>항공권예약관리] 화면에서 다른 결제수단으로 부가서비스를 추가 구매하신 경우 홈페이지에서 항공권 취소 및 부가서비스 개별 취소가 불가하오니 예약센터(1688-8686) 또는 [홈페이지>고객서비스센터>문의합니다] 게시판을 통해 요청해 주시기 바랍니다

Q [항공권] 편의점결제 이용 방법을 알려주세요.

편의점결제 이용 방법은 아래와 같습니다. 1. 항공권 결제 시 [편의점결제] 선택 2. 예매 시 입력한 [예약정보 수신 연락처(E-mail, SMS)]로 편의점 결제 시 필요한 정보 (예약번호, 수납번호) 수신 3. 가까운 편의점(GS25, 7-ELEVEN)에서 예약번호, 수납번호를 제시하고 현금 지불 4. 항공권 예매 완료 ([마이페이지>예약관리>항공권예약관리]에서 예약 내역 확인 가능) 편의점결제 예약 시점으로부터 3시간 이내에 결제하지 않을 경우 예약 신청이 자동으로 취소됩니다.

Q [항공권] 편의점결제 이용 조건을 알려주세요.

편의점결제 이용 조건은 아래와 같습니다. [편의점결제 이용 조건] 1. 출발일로부터 8일 전까지 구매 가능 (7일 이내의 여정은 예매 불가) 2. 국제선 스마트운임, 일반운임 항공권 3. 결제 통화가 KRW인 경우 [편의점결제 불가한 경우] 1. 7일 이내의 항공권 2. 국내선 3. 국제선 이벤트운임 항공권 4. 단체항공권 5. 예약 완료 후 부가서비스 추가 구매

Q [항공권] 익스피디아에서 예약한 항공권 취소가 가능한가요?

익스피디아에서 예약한 항공권의 경우 당사 홈페이지에서 비회원 로그인 후 항공권 여정 변경/취소가 가능합니다. ※ 비회원 로그인 방법 : 마이페이지→항공권 예약관리→ 이메일조회(이메일,예약번호) 또는 탑승자 정보선택(노선선택,탑승자명,예약번호)에 해당 정보 입력 후 비회원 로그인 버튼 클릭

Q [항공권] 인원 수를 줄이고 싶습니다.

(좌석 승인 전) 예약 신청된 상태로, 좌석 승인 전에는 인원 수 조정이 불가합니다. 취소 후 처음부터 다시 예약해주시기 바랍니다. 좌석 판매 현황은 수시로 변경되므로, 재예약 신청 시 운임은 달라질 수 있습니다. (결제 후) 결제 후에 예약을 분리하여 인원수를 줄일 경우에는 구매한 운임에 해당하는 취소수수료가 부과됩니다.

Q [항공권] 단체항공권의 결제시한 연장을 요청하고 싶습니다.

결제 시한 연장은 불가합니다. 결제 시한 내 결제를 하지 않으실 경우 자동 취소되며, 처음부터 다시 예약신청하셔야 합니다. 좌석 판매 현황은 수시로 변경되므로, 재예약 신청 시 운임은 달라질 수 있습니다.

Q [항공권] 단체항공권 좌석 확약을 바로 해줄 수 없나요?

단체항공권 예약 신청 후 접수 순서대로 시스템이 예약가능여부를 처리하므로, 우선 처리는 불가합니다.

Q [항공권] 단체항공권 운임과 좌석 관련해서 담당자 통화하고 싶습니다.

단체항공권 예약 신청 후 영업일 기준 2일 이내에 시스템에 의해 자동으로 예약가능여부를 SMS와 이메일로 안내해 드리오며, 담당자 통화는 불가합니다.

Q [항공권] [나의예약>예약조회]에서 신청한 단체항공권 내역이 보이지 않습니다.

단체항공권 예약을 신청하여 승인 대기 중인 상태와 승인되어 결제 대기 중인 상태인 경우에는 [나의예약 > 예약조회 > 확인 필요]에서 예약 내역 확인이 가능합니다. 결제를 완료한 경우에는 [나의예약 > 예약조회 > 정상]에서 예약 내역 확인이 가능합니다.
