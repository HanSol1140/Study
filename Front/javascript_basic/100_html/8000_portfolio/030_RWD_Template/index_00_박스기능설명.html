1. 로고와 메뉴가 배치 | 모바일 + 태블릿 = 토글 버튼 출력(제이쿼리 사용)
2. 슬라이더 영역 : 사이트 아이덴티티 이미지가 표현되는 영역 #. 가로 전체 너비 사용
3-1. 디스플레이 영역 : 타이틀, 내용이 들어감 모바일 + 태블릿 => 세로배치 | PC => 가로배치
3-2. 프로모션 영역 : 아이콘,타이틀, 간단한 내용, 광고를 위한 영역 흥미를 위해 아이콘등을 배치 => 클릭을 유도 모바일 => 세로배치, 태블릿 | PC => 가로배치
3-3. 최근작업 영역 = cf. 홍보상품, 포트폴리오 : 아이콘, 타이틀, 간단한 내용, 광고를 위한 영역 최대한 잘 보이도록 퀄리티 높은 이미지를 사용 흥미를 위해 아이콘등을 배치 => 클릭을 유도 모바일 => 세로배치, 태블릿 | PC => 가로배치
3-4 . 블로그 영역 = cf. 게시판, 공지사항 등 : 최근에 작성한 글들을 보여주는 영역 모바일 + 태블릿 => 세로배치 | PC => 가로배치
3-5 연락처 영역 : 웹 사이트 관리자에게 연락하기 위한 영역 모바일 + 태블릿 => 세로배치 | PC => 가로배치
4. 푸터 영역 : 웹사이트 저작권, 회사 주소, 회사 지도, 로고 등을 보여줌
