Arduino 노드 설치: node-red를 시작하고 Arduino를 제어하기 위한 Arduino 노드를 설치해야 합니다. 설치 방법은 다음과 같습니다. 좌측 상단의 메뉴 아이콘을 클릭하여 "Manage palette"를 선택합니다. "Install" 탭에서 "arduino"를 검색하고 Arduino 노드를 선택한 다음 "Install" 버튼을 클릭합니다. 설치가 완료되면 "Close" 버튼을 클릭하여 팔레트 관리 창을 닫습니다. Arduino 노드 구성: Arduino 노드를 구성하여 Arduino와의 통신을 설정해야 합니다. 구성 방법은 다음과 같습니다. Arduino 노드를 편집 영역으로 드래그하여 배치합니다. Arduino 노드를 더블 클릭하여 구성 창을 엽니다. "Serial Port" 옵션에서 Arduino가 연결된 포트를 선택합니다. "Board" 옵션에서 Arduino 보드의 종류를 선택합니다. (예: Arduino Uno) "Baud Rate" 옵션에서 통신 속도를 선택합니다. (예: 115200) "Deploy" 버튼을 클릭하여 구성을 저장하고 배포합니다. HTTP 서버 설정: HTTP 서버를 설정하여 웹 페이지를 호스팅하고 웹 브라우저에서 접근할 수 있도록 합니다. 설정 방법은 다음과 같습니다. HTTP 노드를 편집 영역으로 드래그하여 배치합니다. HTTP 노드를 더블 클릭하여 구성 창을 엽니다. "URL" 필드에 /를 입력합니다. "Method" 옵션을 GET으로 선택합니다. "Response" 탭에서 "Response"란에 msg.payload를 입력합니다. "Deploy" 버튼을 클릭하여 구성을 저장하고 배포합니다. 웹 페이지 작성: HTML 웹 페이지를 작성하여 Arduino와 통신하기 위한 사용자 인터페이스를 생성합니다. 웹 페이지의 HTML 코드는 index_html 변수에 이미 포함되어 있습니다. 작성 방법은 다음과 같습니다. Template 노드를 편집 영역으로 드래그하여 배치합니다. Template 노드를 더블 클릭하여 구성 창을 엽니다. "Name" 필드에 적절한 이름을 입력합니다. "Format" 옵션을 Handlebars로 선택합니다. "Template" 필드에 {{index_html}}을 입력합니다. "Deploy" 버튼을 클릭하여 구성을 저장하고 배포합니다. 웹 요청 처리: 웹 페이지에서의 요청을 처리하기 위해 흐름을 구성합니다. 흐름을 구성하는 방법은 다음과 같습니다. HTTP 노드의 출력을 Template 노드의 입력으로 연결합니다. Template 노드의 출력을 Arduino 노드의 입력으로 연결합니다. Arduino 노드의 출력을 Serial Out 노드의 입력으로 연결합니다. Serial Out 노드의 출력을 Debug 노드의 입력으로 연결합니다. "Deploy" 버튼을 클릭하여 구성을 저장하고 배포합니다. 이제 웹 브라우저에서 node-red의 URL로 접속하면 웹 페이지가 표시되며, 입력, 타이머 설정, 포트 변경 등의 기능을 사용할 수 있게 됩니다. 입력, 타이머 설정 및 포트 변경에 대한 요청은 Arduino 노드에서 처리되며, 결과는 Serial Out 노드를 통해 시리얼 모니터에 표시됩니다.
