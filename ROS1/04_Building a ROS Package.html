CMakeLists.txt를 수정해야 할 수 있습니다. 이전 튜토리얼에서 만든 catkin 작업 공간과 새로운 catkin 패키지 beginner_tutorials가 이미 있어야 합니다. 작업 공간으로 이동하고 src 폴더를 확인하세요. cd ~/catkin_ws ls src beginner_tutorials/ CMakeLists.txt@ 정상적으로 생성된 beginner_tutorials 패키지가 확인됬다면 이제 패키지를 빌드할 수 있습니다. catkin_make ls build devel src build와 devel 폴더가 생긴것을 확인할 수 있습니다. build 폴더는 빌드 공간의 기본 위치이며 cmake와 make가 패키지를 구성하고 빌드하는 위치입니다. devel 폴더는 devel 공간의 기본 위치이며, 패키지를 설치하기 전에 실행 파일과 라이브러리가 위치하는 곳입니다. 이제 ROS 패키지를 빌드했으니, ROS 노드에 대해 더 이야기해봅시다.
